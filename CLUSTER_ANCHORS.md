# CLUSTER_ANCHORS - Overnachtingsbases per cluster

Een CLUSTER_ANCHOR is de plek waar Mark bewust meerdere nachten verblijft en van waaruit omliggende locaties worden bezocht. Dit is GEEN A/B/C-locatie en GEEN bestemming uit LOCKED_A of LOCKED_B. Het is een verblijfsbasis.

Onderscheid:
- LOCKED_A en LOCKED_B = plekken waar Mark heen wil (bestemmingen).
- CLUSTER_ANCHOR = plekken waar Mark slaapt (verblijfsbasis).

Routeregel: alle reistijden binnen een cluster worden gemeten vanaf de CLUSTER_ANCHOR, niet vanaf losse locaties.
