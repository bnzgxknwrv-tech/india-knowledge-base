# TOOL CAPABILITY PROTOCOL

For every task requiring external tools, define required operations and test them before substantive execution. Record actual capability, not assumed capability.

A task may proceed under a reduced but explicit method when the protocol permits it. Direct visual map operation, public indexed search, official-link inspection and coordinate parsing are distinct capabilities and produce distinct evidence statuses.

Do not write prompts that demand unavailable operations. When only one candidate requires a Mark-supplied link, isolate that request and continue all unaffected candidates.
