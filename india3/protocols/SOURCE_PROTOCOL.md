# SOURCE PROTOCOL

Each source receives one immutable source ID and records URL or repository path, title, publisher/owner, publication or access date, source type, authority domain, accessibility, limitations and rejection status.

Claims cite source IDs only. A source may support only the domain it actually establishes. Search snippets are discovery evidence, not automatically claim evidence. Reviews may support current experience but not ancient history or institutional identity.

Missing or duplicate source IDs block predecessor transition for material claims. Rejected sources remain recorded with reason to prevent silent reuse.
