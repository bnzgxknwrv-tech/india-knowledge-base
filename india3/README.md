# INDIA 3

INDIA3 is the active architecture for repository-governed India research. A chat is a temporary executor; GitHub is the durable source of truth.

Start every session at `pipeline/ENTRYPOINT.md`. Normal workers read only ENTRYPOINT, NEXT_ACTION, the pinned role contract, the pinned context manifest and its `required_files`.

Core guarantees:
- no automatic Mark A/B/C decisions;
- capability check before task acceptance;
- qualified validation language;
- independent BRONS/ZILVER/GOUD verification;
- complete technical audit plus required user delivery;
- machine-readable recovery and short clean-chat activation.
