# CAPABILITY CHECK

Before external research or artifact generation, record:
- required capability;
- available tool and access mode;
- test performed;
- classification;
- permitted fallback;
- impact on scope.

Classifications:
- `EXECUTABLE_DIRECT`: the required object can be opened and examined directly.
- `EXECUTABLE_PUBLIC_SEARCH`: identity can be investigated through public indexed results, but direct visual operation is unavailable.
- `REQUIRES_MARK_LINK`: one specific unresolved object requires a user-supplied link or file.
- `NOT_EXECUTABLE`: the requirement cannot be met reliably with current tools.

Google Maps visual verification may be claimed only after actual tool-supported place opening and identity matching. Search snippets, copied URLs and coordinate strings do not establish visual verification.

A candidate-level limitation is isolated. Continue all unaffected work unless the run's central conclusion depends on that candidate.
